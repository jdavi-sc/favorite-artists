# 🎵 Favorite Artists

Um projeto desenvolvido para praticar desenvolvimento web consumindo dados relacionados a artistas musicais favoritos. A aplicação apresenta uma interface simples e moderna para exibir informações de artistas, reforçando conceitos de estruturação de páginas, estilização e manipulação de dados no front-end.

O principal objetivo deste projeto foi aprimorar habilidades em desenvolvimento utilizando tecnologias modernas da web, além de fortalecer boas práticas de organização de código e versionamento com Git e GitHub.
